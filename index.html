FAQ

자주 묻는 질문

Q. 랜딩 페이지 호스팅, 도메인 셋업도 해주시나요? +
A.

호스팅과 도메인은 직접 준비해주셔야 합니다.

준비해주신 호스팅에 랜딩 페이지 접속 가능하도록 준비는 가능하지만, 도메인 셋업은 지원하지 않습니다.

별도 비용 추가 시 호스팅 및 도메인 구매·셋업까지 지원 가능합니다.

Q. 로그인, DB, 관리자 페이지 있는 사이트도 만들어주시나요? +
A.

정적 1페이지 랜딩 페이지 전문입니다.

회원가입, 결제, 로그인이 필요하신 경우 협의 후 별도 비용을 추가하여 지원 가능합니다.

Q. 디자인 수정은 몇 번까지 가능한가요? +
A.

랜딩 페이지 : 베이직 2회 / 스탠다드 3회 / 프리미엄 5회

업무 자동화 : 베이직 2회 / 스탠다드 3회

범위를 초과하는 수정은 추가 협의를 통해 별도 산정합니다.

Q. 업무 자동화 시 다른 사이트 정보를 가져올 수 있나요? +
A.

네, 가능합니다.

가능 : 공공 API(나라장터, DART, 공공데이터포털), 공식 RSS, 로그인 불필요한 공개 페이지, 본인 운영 사이트.

거절 : robots.txt 명시 거부 사이트, DB권 보호 정보집약 플랫폼(부동산, 채용, 숙박, 가격비교), 공식 API 제공 사이트의 크롤링 우회.

Q. 랜딩 페이지 및 업무 자동화 운영 관리도 해주시나요? +
A.

제공하지 않습니다.
Victor Lab은 결과물 제작·전달까지만 담당하며, 정기 모니터링 운영은 진행하지 않습니다.

결과물은 고객님 PC 또는 자체 서버에서 직접 실행하시는 형태로 전달드립니다.

Q. 결과물은 어떤 형태로 받나요? +
A.

랜딩 페이지 : 단일 HTML 파일 + 이미지 폴더

업무 자동화 : 실행 파일(Python 스크립트, Excel 매크로 등)과 설치 및 사용 가이드를 전달드립니다.

고객님 PC 또는 자체 서버에서 직접 실행하시는 형태이며, 텔레그램·이메일 알림은 고객님 계정으로 연동해드립니다.

Q. API, 폰트, 이미지 비용은 따로 청구되나요? +
A.

아니오. 외부 API 사용료, 무료 폰트, 무료 스톡 이미지(Unsplash 등)는 작업비에 포함됩니다.

단, 유료 폰트와 유료 이미지는 별도 협의합니다.

Q. 캡차 우회, 자동 로그인, 개인정보 수집은 가능한가요? +
A.

불가합니다.

캡차 우회, 자동 로그인은 사이트 약관 위반, 이메일·전화번호 자동 수집은 정보통신망법 위반 소지가 있어 의뢰 자체를 받지 않습니다.

Q. 시안 받고 진행이 어려워졌어요.
환불되나요? +
A.

단계별 환불 비율을 안내드립니다.

작업 착수 전 100%, 기획·설계 단계 70%, 1차 시안 전달 후 30%까지 환불됩니다.

최종 컨펌 후에는 환불되지 않습니다.

Q. 수정은 몇 번까지 무료인가요? +
A.

시안 단계 2회, 시안 컨펌 후 세부 수정 3회까지 무료입니다.

무료 횟수 초과 시 회당 5만원의 추가 비용이 발생합니다.

단, 단순 텍스트 오타나 문구 수정 같은 경미한 수정은 횟수에 산입하지 않습니다.

Q. 납품 후 문제가 생기면요? +
A.

납품일로부터 7일 이내 발견된 명백한 결함(레이아웃 깨짐, 링크 오류, 명세서 명시 기능 미동작)은 무상으로 수정합니다.

7일 경과 후 또는 신규 요구사항은 별도 견적으로 진행됩니다.

예외 및 세부 조건은 전체 환불 및 작업 정책에서 확인하실 수 있습니다.
