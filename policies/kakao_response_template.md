# 카카오톡 응대용 정책 안내 템플릿

빅터님이 카톡 채널 응대할 때 그대로 복사해서 보낼 수 있는 메시지 모음입니다. 상황별로 사용하세요.

정책 전문: https://lab.victor-jk.com/policies/refund.html

---

## A. 의뢰 확정 시 (견적 합의 후 계약금 안내와 함께)

```
안녕하세요, Victor Lab입니다.
의뢰 확정해주셔서 감사합니다.

▼ 진행 안내
- 작업 시작 전 계약금 입금을 부탁드립니다.
- 작업 단계: 기획 → 1차 시안 → 수정 → 최종 컨펌 → 납품
- 무료 수정: 시안 단계 2회 + 세부 수정 3회 (초과 시 회당 5만원)
- 납품 후 7일 이내 발견된 명백한 결함은 무상 수정해드립니다.

▼ 환불 기준 (요약)
- 작업 착수 전: 100% 환불
- 기획 단계: 70%
- 1차 시안 전달 후: 30%
- 최종 컨펌 후: 환불 불가

전체 환불 및 작업 정책 ↓
https://lab.victor-jk.com/policies/refund.html

위 내용 확인 후 동의해주시면 작업 시작하겠습니다.
```

---

## B. 환불 문의가 들어왔을 때

```
환불 문의 주셔서 안내드립니다.

현재 진행 단계가 [작업 착수 전 / 기획·설계 / 1차 시안 후 / 컨펌 후] 이시네요.
해당 단계의 환불 비율은 [100% / 70% / 30% / 환불 불가] 입니다.

(환불 가능한 경우)
환불 사유와 입금 받으실 계좌 알려주시면 영업일 기준 5일 이내 처리해드리겠습니다.

(환불 불가 경우)
이미 진행된 작업물은 그대로 전달드릴 수 있습니다.
혹시 결과물 수정이나 다른 형태의 마무리가 가능할지 함께 상의드리고 싶습니다.

전체 정책 안내: https://lab.victor-jk.com/policies/refund.html
```

---

## C. 무한 수정 요청이 들어올 때

```
요청해주신 수정 안내드립니다.

이번 요청까지 무료 수정 횟수 [N/M]회가 사용됩니다.
무료 횟수를 초과하는 추가 수정은 회당 5만원으로 진행됩니다.

요청 내용 중 초기 명세서 범위를 벗어나는 항목은 별도 견적으로 안내드릴게요.
모아서 한 번에 전달해주시면 작업 효율이 올라갑니다.
```

---

## D. 납품 후 7일 경과 클레임 들어왔을 때

```
연락 주셔서 감사합니다.

말씀해주신 내용이 명세서에 포함된 기능의 결함이라면 무상으로 수정해드릴 수 있는지 확인해보겠습니다.

다만 납품 후 7일이 지난 시점이라, 신규 요구사항이나 의뢰자 환경 변경(서버 이전, 외부 서비스 정책 변경 등)으로 인한 수정인 경우 별도 견적으로 진행될 수 있습니다.

증상을 자세히 알려주시면 검토 후 안내드리겠습니다.
```

---

## E. 단순 변심·프로젝트 중단 통보 시

```
안타깝지만 의뢰자 사정으로 중단되시는 경우, 진행 단계까지의 작업 비용은 환불이 어려운 점 양해 부탁드립니다.

지금까지 진행된 작업물(시안, 기획안 등)은 정리하여 전달드릴 수 있습니다.
필요하신 형태로 알려주세요.

자세한 정책: https://lab.victor-jk.com/policies/refund.html
```

---

## 사용 시 주의

- 메시지 안의 `[N/M]` 같은 placeholder는 실제 수치로 교체하세요.
- 환불 비율 안내 시, 결제된 금액(분할 결제면 입금 시점까지의 누적액) 기준임을 명시할 수 있습니다.
- 크몽 의뢰 건은 크몽 자체 환불 규정이 우선 적용됨을 한 줄 추가하세요.
